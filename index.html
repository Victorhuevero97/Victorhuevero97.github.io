HTML

HTML, siglas de HuperText Markup Language, hace referencia al lenguake de marcado para la elaboración de páginas web. Es un estándar que sirve de referenciapara la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes entre otros. Es un estándar a corgo de la W3C, organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo refetente a su escritura e interpretación. Es el lenguaje con el que se defienden las páginas web.

Fuente: HTML. Wikipedia.

HTTP

Hupertext Transfer Protocol o HTTP (en español protocolo de trasferencia de hipertexto) es el protocolo usado en cada trasacción de la World Wide Web. HTTP fue desarrollado por el World Wide Web Consortium y la Internet Engineering Task Force, colaboración que culminó en el 1999 con la publicación de una serie de RFC, el más importante de ellos es el RFC2016 que especifica la versión 1.1

Fuente: HTTP. Wikipedia

URL

Un Localizador de recursos uniforme o URL -siglas en ingés de Uniform Resorce Locator- es un didentificador de recursos uniforme (URI) cuyos recursoso referidos pueden cambiar, esto es, la dirección puede apuntar a recursos variables en el tiempo. Están formados por una secuencia de caracteres, de acuerdo a un formato modélico y estándar, que designa recursos en un red., como Internet.
